## Usage
How to install and use this application:

```php
//Installing
git clone https://github.com/dev-techguy/Laravel-Universe

//copy the contents
cp .env.example .env

//Run this to seed datatbase
php artisan migrate:refresh --seed

//Then
php artisan serve

```
